Additional Input
- additional input ini melekat pada sub kategori
- punya additional input, jumlah tak terbatas
- additional input bisa berupa text, select, checkbox
- additional input bisa punya child, dan seterusnya
- additional input punya properti untuk fungsi validasi seperti input text bisa punya maxlength 40 dkk
- referensi
  - http://www.tokobagus.com/pasang.asp
  - http://www2.mudah.my/ai/form/1
- contoh kategori mobil
  - ada additional input "merk mobil" dengan pilihan : pilihan honda, toyota, suzuki
  - jika dipilih honda maka akan keluar additional input jenis mobil dari merk honda saja
  - contoh detail : http://www.tokobagus.com/mobil/baru/
- halaman masing2 sub kategori akan berbeda2 karena filternya juga berbeda2

Filter Harga Pencarian
- setiap sub kategori akan memiliki filter range harga yang bisa di masukan oleh admin
- terdiri dari 2 filter (harga min & harga max)

Verify User
- ada 2 jenis Verify (email & lokasi)
- di admin ada fitur untuk get verify location yang akan mengenerate no yang no tersebut akan
  dikirim admin ke lokasi si request via pos dan si request akan memasukan no tersebut untuk melakukan verifikasi
- batas untuk verify lokasi sebanyak 3x, jika sudah lebih dari 3x gagal maka user tersebut tidak bisa melakukan verifikasi lagi

Validasi
- sebelum memasang iklan, user harus mengisi profile terlebih dahulu dengan lengkap, jika belum di isi maka user tidak bisa mengirim iklan dia

Follow Publisher
- jika ada yang follow penjual, ketika penjual tersebut memasang iklan, maka user yang follow penjual tersebut mendapat informasi email
- pengiriman emailnya ketika iklan yang dipasang penjual telah di approve oleh admin dan dilakukan oleh cron
- follow ini merupakan button di profile penjual, jika ada yang akan follow tetapi dia belum login, maka diarahkan ke form registrasi

Session Login
- session ada umurnya (2 jam)
